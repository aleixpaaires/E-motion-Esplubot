# Wassily Kandinsky

## 1. Tipo de trazo

- Líneas rectas y curvas de distintos grosores.
- Círculos, arcos, triángulos y otras formas geométricas.
- Superposición de elementos pequeños y grandes.
- Combinación de geometría precisa con formas más libres o biomórficas.

## 2. Movimiento visual

- Rítmico y organizado, parecido a una composición musical.
- Alterna movimientos cortos y largos.
- Puede pasar de zonas tranquilas a zonas con mucha actividad.
- Mantiene equilibrio entre orden geométrico y expresión.

## 3. Uso del color

- Contrastes fuertes entre colores vivos y negro.
- El color funciona como elemento expresivo, no como representación realista.
- Para E-motion: rojo, amarillo, naranja, azul y violeta pueden formar el contenido principal; negro puede definir líneas y contrastes.

## 4. Composición

- Distribución abstracta basada en relaciones entre color, forma y línea.
- Uso frecuente de geometría, especialmente en su etapa vinculada a la Bauhaus.
- Elementos de distintos tamaños crean ritmo y jerarquía.
- La composición puede organizarse como una secuencia visual de notas musicales.

## 5. Traducción a brazo robótico

- Dibujar líneas rectas, diagonales, arcos y círculos separados.
- Alternar trazos negros finos con formas coloreadas.
- Usar velocidad media para geometría y velocidad baja para círculos precisos.
- Aplicar presión suave o media; evitar presión fuerte antes de calibrar el pincel.
- Distribuir formas con márgenes claros para evitar colisiones y exceso de pintura.

## 6. Ideas de funciones robóticas

- `kandinskyLines()`
- `kandinskyCircles()`
- `kandinskyArcs()`
- `kandinskyGeometricRhythm()`
- `kandinskyContrastMarks()`
